### Home Page

- Displays the latest 100 news stories in a list, sorted by date with the freshest at the top.

  - Title
  - Rating
  - Author's nickname
  - Publication date
- Click to navigate to the news page
- Automatic news list refresh every minute
- Button for manual news list refresh
